# Paint application

## This is a paint application

## How to use paint?

#### **On Keyboard**
#### Choose colors

* 1 - choose blue mode
* 2 - choose green mode
* 3 - choose red mode

#### Choose figures

* c - choose to draw circle
* r - choose to draw rectangle
* e - choose to use eraser
* p - choose pencil(brush)
* l - choose to draw line
* s - choose to draw square
* t - choose to draw right triangle
* q - choose to draw equilateral triangle
* d - choose to draw rhombus
* f - choose to use fill tool
* m - to type text

### Choose brush size and width
* F1 - 2 (small size)
* F2 - 5 (medium size)
* F3 - 10 (large size)

### Save canvas
* Press CTRL+S to save the canva in PNG format

## Use your mouse to manage the position of drawing

#### if you press the left mouse button you will draw one figure and you can hold button pressed to draw some figure using the initial figures
