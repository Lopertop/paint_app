# Paint application

## This is a paint application

## How to use paint?

#### **On Keyboard**
#### Choose colors

* 1 - choose blue mode
* 2 - choose green mode
* 3 - choose red mode

#### Choose figures

* c - choose to draw circle
* r - choose to draw rectangle
* e - choose to use eraser

## Use your mouse to manage the position of drawing

#### if you press the left mouse button you will draw one figure and you can hold button pressed to draw some figure using the initial figures
